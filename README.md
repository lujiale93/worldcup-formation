# ⚽ World Cup 2026 Formation Lab

Build and share football formations for the FIFA World Cup 2026.

## Features
- 10 national teams (Brazil, France, England, Germany, Spain, Argentina, Portugal, Netherlands, Japan, South Korea)
- 5 formations: 4-3-3, 4-4-2, 4-2-3-1, 3-5-2, 5-3-2
- Drag & drop players onto the pitch
- Player stats & form viewer
- Auto-fill formation

## Local Development
```bash
npm install
npm run dev
```

## Deploy to GitHub Pages
Push to `main` branch — GitHub Actions deploys automatically.
